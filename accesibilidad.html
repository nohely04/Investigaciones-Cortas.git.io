¿Qué es la accecibilidad web?

El objetivo de la accecibilidad web es conseguir que las páginas web puedan ser utilizadas por el máximo número de usuarios, independientemente de sus conocimientos o capacidades personales, y de las características técnicas de los dispositivos utilizados para acceder a Internet.

Si bien, la necesidad de que la web sea universal y accesible a cualquier persona, era un requisito primordial para su creador Tim Berners-Lee, de aquí su frase:

"The power of the Web is in its universality. Access by everyone regardless of disability is an essential aspect."

"El poder de la Web está en su universalidad. El acceso por cualquier persona, independientemente de la discapacidad que presente es un aspecto esencial."

Importancia de la accecibilidad web

La accesibilidad web es esencial para que las personas con discapacidades puedan acceder y usar el contenido en línea de manera efectiva. Además, los sitios web accesibles ofrecen una mejor experiencia de usuario para estas personas.

Por eso, es importante conocer los 3 tipos de accesibilidad y cómo acceder a la web. Las cuales son:

1. Accesibilidad sensorial

Esto abarca tanto la discapacidad auditiva como la visual. Por eso en nuestra web debemos de implementar alternativas para todas aquellas personas que no pueden escuchar o no pueden ver tengan la posibilidad de ingresar a la web y sentirse bien en las diferentes páginas web a su alcance .

2. Accesibilidad física

Esta se da cuando falta alguna parte del cuerpo, o bien, cuando una parte del cuerpo no puede realizar su función. Para esto debemos de implementar en nuestra web diferentes acciones utilizando una tecla para aquellas personas que navegan a través de enlaces.

3. Accesibilidad cognitiva

Esta discapacidad es para toda la vida y presenta una serie de limitaciones en las habilidades diarias de la persona, entre ellas están una baja compresión lectora, dando por resultado la complicación para aprender y comunicarse. Por esto, en nuestra web debemos de implementar en darle más tiempo de respuesta y ofrecerles una lectura más fácil.

Principios de la accecibilidad web

La accesibilidad web tiene 4 principales principios, los cuales son:

Perceptible

Este principio enfatiza la necesidad de que el contenido de la página web y su interfaz sea perceptible para todos los usuarios. Para esto se debe de ofrecer alternativas a los contenidos no textuales, ya sean imágenes, vídeos, íconos, etc. Ejemplo:

Si una página web utiliza un ícono para el botón de agregar productos al carrito una alternativa es colocarle una etiqueta al botón para que pueda ser fácil de leer y comprender.

Operable

Este principio se refiere a que un sitio o página web el usuario debe poder interactuar con la interfaz sin ninguna complicación, por esto, debe de ser fácil su navegación y con suficiente tiempo para interactuar. Ejemplo:

Una página o sitio web debe de estar acondicionada para que se pueda usarse con el teclado, esto ayuda a los usuarios con poca movilidad física sea fácil su navegación

Comprensible

Este principio es importante para todos los usuarios, ya que el contenido debe de ser fácil de leer y comprender, las páginas deben de ser predecibles y utilizar los estándares establecidos en la navegación. Todo esto con el fin de que cualquier entre al sitio o página web pueda tener una buena experiencia de usuario. Ejemplo:

Si una página web para ventas de tiquetes, debe de dar tiempo para que el usario llene los espacios solicitados con sus datos, más si son datos sensibles ayudando así a evitar errores..

Robusto

Este principio nos indica que el contenido debe de ser seguro y bien desarrollado de una forma confiable para que pueda utilizar varias aplicaciones de usuario, incluyendo las ayudas técnicas. Ejemplo:

Si una página o sitio web se encuentra el inglés esta debe de tener la opción de traducir su contenido para que así los usuarios puedan navegar en ella.
